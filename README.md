## react16-mobx

> demo react16 react-router(v3|v4) + mobx


### 项目启动



```
git clone https://github.com/pplgin/react16-mobx.git

cd react16-mobx

// install node_modules
yarn

// start demo
yarn dev

open http://0.0.0.0:9091
```


Default is router-v4, If you use router-v3, checkout branch `router-v3`
